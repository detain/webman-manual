# 接入webman-admin

[webman-admin](https://www.workerman.net/plugin/82) 是webman官方開發的管理後台。支持權限設置、菜單生成(一鍵curd)，自帶一個用戶表，方便各個應用插件共享用戶數據。

## 接入webman-admin好處

* 為各個應用插件提供統一的後台入口，提升用戶體驗
* 減少應用插件後台開發時間，降低維護成本
* 管理後台自帶插件管理功能，方便插件的集中檢索、安裝
